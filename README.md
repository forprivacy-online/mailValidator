# 📬✅ mailValidator


### ⭐ Features:

- Validates email looks like an email i.e. contains an "@" and a "." to the right of it.

- Validates MX records are present on DNS.

- Validates SMTP server is running.

- Validates mailbox exists on SMTP server.
